# linux-bits
Repository contiaining CLI tools and scripts. On my machine, I alias them for ease of use.

## Number Base Converter
A python script that takes a number in decimal, hexadeximal, octal, or binary and ouputs the value in the other bases. Format can be specified 0x__ or __h, 0b__ or __b. Multiple numbers can be entered space delimited.
![ascii_table_print](https://github.com/JamesGWatson/linux-bits/blob/main/screenshots/number%20base%20converter/number_base_converter_single.png?raw=true)

## ASCII Table Print
Display the ASCII table and codes. Command line arguments offer options such as slim formatting, printable characters only, and no decimal value outputs.
